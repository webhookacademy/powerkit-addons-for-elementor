== Installation Guide ==

Follow these steps to install Powerkit Addons For Elementor on your WordPress website:

1. **Download the Plugin:**
   - Download the `powerkit-addons-for-elementor.zip` file.

2. **Upload the Plugin:**
   - Log into your WordPress admin dashboard.
   - Navigate to `Plugins` > `Add New`.
   - Click `Upload Plugin` and select the `powerkit-addons-for-elementor.zip` file.

3. **Activate the Plugin:**
   - Once the installation is complete, click `Activate` to enable the plugin.


== Frequently Asked Questions ==

= Do I need Elementor Pro to use this plugin? =
No. PowerKit Addons for Elementor works perfectly with the free version of Elementor. However, if you have Elementor Pro, you can use our addons alongside Pro widgets and features without any conflict.

= Why are some templates not importing? =
If a template contains many images or large media files, it may take longer to import depending on your hosting speed and PHP settings. If the import fails, try:
1. Increasing your PHP memory limit and max execution time.
2. Importing again after clearing your browser cache.
3. Ensuring you are using the latest version of Elementor and this plugin.

= Will this plugin slow down my site? =
No. PowerKit Addons for Elementor is built to be lightweight and optimized for performance. All assets are loaded only when needed to avoid unnecessary bloat.

= Can I use this plugin on client websites? =
Yes. You can use PowerKit Addons on unlimited personal or client websites, as per the GPL license.

= Is it compatible with my theme? =
Yes. This plugin works with any theme that supports Elementor, including popular themes like Astra, OceanWP, and GeneratePress.

= How often do you release new templates or widgets? =
We regularly release updates with new widgets, templates, and features to keep your designs fresh and your workflow efficient.
